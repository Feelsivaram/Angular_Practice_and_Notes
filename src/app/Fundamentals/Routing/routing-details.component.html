Routing details

Step 1) We need to implement import RouterModule from '@angular/router' in app.module.ts file. So, that we can navigate the pages.

Step 2) We need to implement router-outlet tag in HTML pages where the child pages need to navigate/implements.

Step 3) Add routerLink attribute where the page should navigate in HTML tags. (ex: routerLink="Home")
